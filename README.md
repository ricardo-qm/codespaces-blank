# Calculadora AI

Este proyecto es una calculadora basada en inteligencia artificial desarrollada en Java.

## Descripción

El proyecto "calculadora-AI" tiene como objetivo proporcionar una calculadora avanzada que utilice técnicas de inteligencia artificial para mejorar la precisión y eficiencia en los cálculos.

## Estructura del Proyecto

La estructura del proyecto incluye los siguientes directorios y archivos:

- **src/**: Contiene el código fuente del proyecto.
- **test/**: Contiene las pruebas unitarias para el proyecto.

## Instalación

Para instalar y ejecutar este proyecto en tu máquina local, sigue estos pasos:

1. Clona el repositorio en tu máquina local:
   ```sh
   git clone https://github.com/ricardo-qm/calculadora-AI.git
   ```

2. Navega al directorio del proyecto:
   ```sh
   cd calculadora-AI
   ```

3. Compila el proyecto utilizando Maven:
   ```sh
   mvn compile
   ```

4. Ejecuta los tests para asegurarte de que todo funciona correctamente:
   ```sh
   mvn test
   ```

## Uso

Para utilizar la calculadora, ejecuta la clase principal del proyecto desde tu entorno de desarrollo o utilizando la línea de comandos:
```sh
java -cp target/calculadora-AI-1.0-SNAPSHOT.jar com.example.Main
```
